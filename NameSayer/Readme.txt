

==========================How to Run=================================

 Open the terminal and cd to the directory with the jar file
 make sure that there is a folder called 'names' in the directory 
 which contains the database names. Then type 

	          "java -jar NameSayer.jar" 

 to run the jar 
 
=====================================================================



==========================How to Use=================================

1. Click on "Listen to Name(s)" to view all names in the database
   and then select the versions of the names you want to practice
   and click next. Clear button can be pressed to clear all of the
   Selected names.

2. Select the name and press "Play" to listen to the database 
   recording. Press shuffle to randomise the play order. To select
   the next name in the list either click or press "Previous" or 
   "Next".

3. If the quality of the recording is bad, then press the quality
   toggle button state that the quality is bad. All of the recordings
   that are marked as bad will be in the "Bad_Recordings.txt" file.

4. To practice a name click on "Practice Name" and it will show a 
   pop up window that will contain the following
 
	1) Record your attempt
	2) Listen to attempt
	3) Compare Attempt*
	4) Save Attempt

   *compare plays recorded attempt first and then database name
   
   Record will record your attempt for a certain name. Listen will
   play back the recording of your attempt. Compare will play your
   recording first and then the recording of that same name from
   the database. Save will save your the recording of your attempt
   and the saved recording will be present in after clicking 
   "Show Previous Practices" from the main menu.

5. To access all previous practices press the respective 
   "Show Previous Practices". This will then show you all the 
   practices.

6. If you click on a practice name you can see all the different
   versions of the database name and play both with the respective 
   buttons. you can also delete the user attempts with the delete 
   button.

7. To test if your mic is working click test mic on main menu
